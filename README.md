# Cadence Virtuoso Schematic Shortcuts Cheat Sheet 🚀

A beautifully designed, bilingual (English & Arabic), and mobile-responsive cheat sheet for Cadence Virtuoso Schematic Editor keyboard shortcuts. 

## 🌟 Overview

When working on Integrated Circuits (IC) and schematic design in Cadence Virtuoso, mastering keyboard shortcuts is essential to speed up your workflow. This repository hosts a neat, printable, and mobile-friendly HTML cheat sheet that lists the most common and useful shortcuts.

### **[🔗 View the Live Cheat Sheet Here](https://omarafifi-cse.github.io/cadence-virtuoso-shortcuts/)** 

## ✨ Features

- **📱 Mobile Responsive:** Perfect for viewing on your phone while working on the desktop. The layout adapts into a clean card-based view on small screens.
- **🖨️ Print-Ready:** Specifically styled to omit unnecessary UI elements when printed, creating a clean physical cheat sheet.
- **🌍 Bilingual Support:** Clear explanations of what each shortcut does in both English and Arabic.
- **🎨 Modern UI:** Sleek and readable visual design.

## 🛠️ How to Host on GitHub Pages

Since the entire cheat sheet is contained within a single `index.html` file, deploying it is incredibly fast:

1. Create a new public repository on GitHub (e.g., `cadence-virtuoso-shortcuts`).
2. Push the contents of this repository to GitHub.
3. Go to your repository's **Settings** > **Pages**.
4. Under "Build and deployment", set the **Source** to `Deploy from a branch`.
5. Select the `main` (or `master`) branch and click save.
6. Your cheat sheet will be live at `https://<your-username>.github.io/<repo-name>/`!
